Systems

Examples of systems I can deliver

These examples show the kind of work I like to focus on: clear responsibilities, readable, typed code and systems you can expand over time.

System AI · Pathfinding
Boss / advanced NPC AI
NPC that chases players using Pathfinding, manages distance checks and switches between multiple attacks or behaviours depending on health or timers.
• Server-side logic with configuration modules for tuning
System Data · Profiles
Player profiles & data saving
Profiles that store stats, currency and advenced inventory data, using safe DataStore patterns, pcall and typed APIs so other scripts can easily read/write player info.
• Good starting point for larger progression systems
System Dialogue · UI
Dialogue & interaction framework
NPC dialogue built from data tables, with choices, simple branching and UI that stays in sync with server logic.
• Data-driven, easy to extend with more NPCs and quests
System Utility
Small tools, fixes & refactors
Cleaning up existing scripts, reorganizing them into modules, fixing bugs and adding small tools (debug, admin, QoL systems) to keep your project maintainable.
• Ideal if you already have a game but need a scripter to refine it
